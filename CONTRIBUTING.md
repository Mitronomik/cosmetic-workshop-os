# Contributing

One PR equals one roadmap step.

Workflow:
1. Read `AGENTS.md`.
2. Read `docs/architecture.md` and `docs/roadmap.md`.
3. Read `state/current-focus.md`.
4. Implement only the scoped task.
5. Add or update tests.
6. Run checks and smoke if applicable.
7. Update `state/progress.md` and `state/handoff.md`.

Required PR summary:

```markdown
## Summary
## Scope
## Data model / migrations
## User-visible changes
## Tests
## Risks / limitations
## Follow-up
```
