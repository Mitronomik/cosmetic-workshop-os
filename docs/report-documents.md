# Документы отчётов

Документы отчётов позволяют создать отдельный локальный файл «Сводка мастерской» по текущим данным отчётов.

## Что поддерживается сейчас

- Markdown-файл (`.md`);
- PDF-файл (`.pdf`), если приложение находит подходящий локальный шрифт для русского текста;
- явное создание документа пользователем;
- список уже созданных документов;
- безопасное открытие или скачивание известных Markdown/PDF-файлов из приложения;
- добавление заполненных полей профиля мастерской в новые документы.

## Что не поддерживается

- DOCX;
- графики;
- счета, акты, этикетки, сертификаты, технологические карты рецептов, бухгалтерские и налоговые документы;
- автоматическое создание по расписанию;
- отправка по почте или в облако;
- редактор шаблонов;
- загрузка произвольных файлов через этот экран.

## Безопасность

Документ создаётся только после явного нажатия в `/report-documents`. Открытие отчётов или списка документов не создаёт новые файлы.

Создание документа:

- берёт текущую сводку отчётов;
- добавляет display-only поля профиля мастерской, если они заполнены;
- не меняет рецепты, клиентов, заказы, склад, производство, алерты, закупки, импорт, демо-данные или журнал действий;
- не создаёт резервную копию;
- не создаёт экспорт данных;
- не меняет уже созданные Markdown/PDF-документы.

## Где лежат файлы

Файлы сохраняются локально в папке:

```text
<папка данных мастерской>/exports/report-documents/
```

Имена файлов создаёт приложение. Пользователь не вводит имя файла вручную, а существующие файлы не перезаписываются.

## Содержимое документа

Документы на русском языке и включают:

- время создания;
- время формирования исходной сводки;
- заполненные поля профиля мастерской;
- сводку склада;
- сводку заказов;
- сводку производства;
- алерты и закупки;
- базовую финансовую сводку;
- предупреждения о неполных данных;
- пояснение, что документ не является бухгалтерским или налоговым отчётом.
