# Импорт CSV/XLSX через черновики

Импорт в **Мастерской косметолога** всегда проходит через безопасный черновик.

## Безопасный поток

```text
загрузка файла → черновик → предпросмотр → проверка → явное подтверждение → применение для поддерживаемых типов
```

Строки файла никогда не записываются в рабочие данные сразу после загрузки. Пользователь сначала видит предпросмотр, ошибки и предупреждения. Применение выполняется только после явного подтверждения.

## Поддерживаемые файлы

- `.csv`
- `.xlsx`

Не поддерживаются для импорта:

- PDF;
- изображения (`.png`, `.jpg`, `.jpeg`, `.heic`);
- `.docx`;
- `.txt`;
- OCR или AI-извлечение.

## Ограничения

- Максимальный размер файла: 5 MB.
- Максимум строк данных: 5000.
- Максимум столбцов: 100.
- Для XLSX читается первый видимый лист.

## Поддерживаемые цели

| Тип | Что нужно минимум | Что можно добавить |
| --- | --- | --- |
| Компоненты | `name` / `Название` | INCI, единица, плотность, заметки |
| Тара | `name` / `Название` | категория, единица, стоимость, остаток, минимум |
| Клиенты | `full_name` / `ФИО` | телефон, email, адрес, заметки |
| Рецепты | `name` / `Название` | тип продукта, заметки |
| Партии компонентов | `ingredient_name`, `quantity`, `unit` | цена, даты, поставщик, номер партии |
| Заказы | `client_name`, `product_name`, `target_batch_size_value`, `target_batch_size_unit` | цена продажи, срок, заметки |

## Что можно применить сейчас

Применение в рабочие данные поддержано только для безопасных целей:

- компоненты;
- клиенты;
- карточки рецептов без состава;
- тара без движения остатков.

Партии компонентов и заказы остаются черновиками: их можно проверить, но нельзя применить в рабочие данные с текущего экрана.

## Проверки

Импорт проверяет обязательные столбцы, пустые значения, числа, даты, единицы измерения, дубликаты, неизвестные столбцы и другие проблемы. Ошибки указывают строку, поле, значение и понятное действие, когда это возможно.

Безопасные русские заголовки распознаются, например `Название`, `ФИО`, `Компонент`, `Дата_покупки`, `Цена`. Десятичная запятая вида `100,5` может быть нормализована в `100.5`, а неоднозначные форматы чисел блокируются.

## Перед применением

Перед применением импорта рекомендуется создать резервную копию. Приложение не создаёт её автоматически.

Если файл выглядит неверно, исправьте исходную таблицу и загрузите новый черновик. Существующие записи не обновляются автоматически; при конфликте применение отменяется полностью.
